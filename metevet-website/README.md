# MeteVet Website

Premium, responsive, static veterinary clinic website.

## Files
- index.html
- assets/style.css
- assets/script.js

## Before publishing
Replace these placeholders:
- +90 500 000 00 00
- 905000000000 (WhatsApp links)
- Clinic address
- Working hours
- Doctor photo/clinic photos if available

## Publish
You can upload the full folder to:
- cPanel public_html
- Netlify
- Vercel
- GitHub Pages

The appointment form currently opens WhatsApp with a pre-filled message.
